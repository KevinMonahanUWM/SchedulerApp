Acceptance Test for Admin Delete Course


Scenario: Admin successfully deletes an existing course
Given the admin is logged in to the system
And there is at least one existing course
And the admin has navigated to the “Delete Course” page
When the admin selects a course to delete
And the admin confirms the deletion
Then the course should no longer be present in the system
And all related sections and user assignments should be removed
And a confirmation message “Course successfully deleted” should be displayed

Scenario: Admin attempts to delete a course when there are none available
Given the admin is logged in to the system
And there are no courses to delete
When the admin navigates to the “Delete Course” page
Then a message “No Courses to Delete” should be displayed
